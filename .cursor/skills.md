## Core Competencies for R Package Development

* **Advanced R Programming:** Proficiency in base R, functional programming paradigms, environments, and namespace management.
* **Data Wrangling (data.table):** Expert-level syntax and optimization using `data.table`. Deep understanding of update-by-reference semantics (`:=`) and Non-Standard Evaluation (NSE) management to prevent scoping issues.
* **Package Architecture:** Comprehensive understanding of the standard R package directory structure (`R/`, `man/`, `tests/`, `vignettes/`, `NAMESPACE`, `DESCRIPTION`).
* **Development Ecosystem:** Fluency in the standard toolchain for R package development, specifically `devtools`, `usethis`, `roxygen2`, and `testthat`.
* **Modular Design:** Ability to break down complex processes into small, pure, and highly testable helper functions.
